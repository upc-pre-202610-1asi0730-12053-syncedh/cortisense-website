# CortiSense Website

Landing page basada en la estructura de VitalWatch, adaptada a CortiSense/SyncedHealth.

## Probar localmente

Abre `index.html` directamente o ejecuta:

```bash
python -m http.server 5500
```

Luego entra a:

```text
http://localhost:5500
```

## Links de la Web App

Los botones de Acceder y Elegir plan apuntan a:

```text
https://upc-pre-202610-1asi0730-12053-syncedh.github.io/cortisense-webapp
```

Si tu Web App final tiene otra URL, reemplaza esa cadena en `index.html` y `terms.html`.

## Deploy

No se incluye workflow de GitHub Actions de VitalWatch para evitar copiar credenciales o configuración de otro proyecto.
